# Project Rules

1. **Package Manager**: Always use `bun` for installing dependencies and running scripts.
   - Install: `bun install`
   - Run: `bun run <script>`
   - Exec: `bunx <command>`
   - Do NOT use `npm`, `yarn`, or `pnpm`.

2. **Tech Stack**:
   - Cloudflare Workers (Hono, Durable Objects).
   - TypeScript.
   - Turborepo.
